# Oscilloscope

집에서 회로를 작성하다가, 집에 오실로스코프가 하나쯤은 있으면 편하겠다는 생각이 들었다. 그런데, 오실로스코프가 너무 비싸다...

그래서 회로도 몇개 연습해 볼겸 오실로스코프 제작을 시작했다.
